# AI Learning System

> This file is the persistent memory layer for your AI assistant.
> LearnBoard reads and edits this file through a web UI.
> The AI reads this file at the start of each session to remember your preferences.

## How it works

After significant work sessions, the AI updates the tables below with:
- **User Requests** — tools and features you asked to build (strongest signal)
- **Suggestions Log** — ideas proposed, with a vote and implementation status
- **Lessons Learned** — mistakes and corrections the AI should not repeat
- **AI Observations** — patterns noticed across sessions
